# Home Credit Default Risk Prediction - MLOps Project

## Project Overview
End-to-end Machine Learning project to predict whether a client will have payment difficulties (loan default) using Home Credit Default Risk dataset (307k+ rows).

## Features Demonstrated
- Complete ML Pipeline
- MLflow for Experiment Tracking & Model Versioning
- Automated Preprocessing & Feature Engineering
- Model Training + Hyperparameter Tuning
- FastAPI Deployment
- GitHub + Proper Version Control
- MLOps Practices
